
# Maintenance Management System

A full stack application that keeps track of a truck shops customer, maintenance, and employee information.


## Authors

- [@GabCar95](https://www.github.com/GabCar95)


## Features

- User authentication
- Displays realtime data
- CRUD


## Frontend

 - React 
 - TailwindCSS
 - Axios

## Backend
- Django
- Django REST Framework
